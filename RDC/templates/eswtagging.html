Follow these steps to check ESW tagging

• Login to ibmrpcw0

• sudo su – wasrpcw0

• Check if the service is running with

ps -ef | grep Tag

• Check the logs with

tail -n50 /home/wasrpcw0/TagChildren/TagChildrenBuilderLog.log

• You'll see messages like this:

******************************************************************

Run Time:Fri Nov 05 17:48:19 GMT 2010, DETAILS: PROCESSID: 2010-11-05-17.48.20.000000,STATUS: SUCCESFUL WITH NO ERRORS, TAG REQUIRED:3231, TAGGED:3231, ERRORS:0,ERROR DESC:{}

• You can come across this kind of message as well:

******************************************************************

Run Time:Fri Nov 05 08:31:56 GMT 2010, DETAILS: PROCESSID: 2010-11-05-08.31.56.000000,STATUS: ESW_Bean:getLastRunTS():- Got NULL Last Run Timestamp! ,ERROR DESC:{}

It's just a warning. It means there was nothing to process at the time.
