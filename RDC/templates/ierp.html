Follow these steps to check IERP

• Go on ibmrpcg0 server, under rdtrpcg0 id, and check whether the crontab is uncommented (if no reason to be commented it must be uncommented)

• Confirm whether the Extractor job processed successfully on last run (it runs on each 10min) with:

tail -f /ierp/extr/traceinfoIERPExtractorRealTime.out

• Confirm whether the Publish job processed successfully on last run with:

tail -f /ierp/xmlpub/traceinfoIERPDataXMLGenPublish.out

check generated xml

Go to g00server with sudo su – rdtrpcg0

then go to the path

cd /ierp/xmlpub/xmldata/processed

check with the command ls -ltr|tail -10 the last time and date of generated xml.

check the current date and Timestamp by using date command.

If the processed file is of current date and time then its fine

If not, then please go to the path cd /ierp/xmlpub/xmldata and check with the below command that publisher is running or not.

If its running and still files are not generated then please kill the publisher and restart again. And again check the generated xml.

ps -ef|grep -i ierp|grep -v mqm

to kill the publisher : kill -9 id
