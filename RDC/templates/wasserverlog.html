Follow these steps to check WAS logs on W11/W22/W33 servers.


W11


1) login to w11 in user wasrpcw11

2) Goto below path:

/usr/WebSphere_rpcw1/AppServer/profiles/serverejb01/logs/server1

3) ls -ltr

4) tail -f SystemOut.log

Please check that the log should be of current date as below:


And the log should not contain any error message or stopped message like below:



W22


1) login to w11 in user wasrpcw22

2) Goto below path:

/usr/WebSphere_rpcw2/AppServer/profiles/serverejb01/logs/server1

3) ls -ltr

4) tail -f SystemOut.log


W33


1) login to w33 in user wasrpcw33

2) Goto below path:

/usr/WebSphere_rpcw3/AppServer/profiles/serverejb01/logs/server1

3) ls -ltr

4) tail -f SystemOut.log

Please follow the same process for all three remaining servers.

If the server is stopped, please contact VCS team to restart the server
