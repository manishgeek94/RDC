Follow these steps to check translation services

# CI Translation Service allows an application to convert from a Site Party Id to a CMR (Outbound from iERP) and from CMR to a Site Party Id (Inbound to iERP). In addition there are operations exposed within the Translation Service for admin purposes (create, update, delete, find, etc). The Admin functions can also be utilized via the RDC GUI/Find CMR.

# The service implementation consists of a JAX-RS (REST) endpoint which intercepts HTTP messages (POST for actions and GET for status messages) and processes the XML content sent within the HTTP message for different functions.

# For more details on the individual operations (and XML formats) consult the Web Services Wiki (Translation Services) at this link:

https://w3-connections.ibm.com/wikis/home?lang=en-us#/wiki/W9636a2b06f44_406a_8159_214bf99bed98/page/REST%20Operations

Verifying the Translation Service is active and responding to Requests


To test that the Translation Service EAR is active and processing incoming requests, simply paste the environment specific endpoint URL into a browser (which by default sends an HTTP GET request to the endpoint). If active, the Translation Service will return a message saying the following: ‘Hello, This is a Translation REST Service!’.

Testing a Translation Service translation with the Firefox Poster Plugin


To test the Translation Service you must have an HTTP Client which can send HTTP POST messages along with XML content. There are several of these available on the internet, and the CI Development team recommends the Poster Plugin/Add-on for Firefox due to its simplicity in usage.


Installing the Poster Firefox Add-on Plugin


To download and install the Poster Plugin Add-on:

1. Select Tools->Add-ons from the Firefox browser menu.

2. Type in ‘Poster’ in the search bar at the top of the screen. Click the Search button icon.

3. Scroll until you find the Poster plugin, and click “Add to Firefox’. Follow the on-screen instructions to complete the installation.

Using the Poster Firefox Add-on Plugin

To test with the Poster plugin, select Tools -> Poster from the Firefox browser menu. A new window will appear similar to the following:


Fill in the URL field with one of the Translation Service endpoints. Typically use the load balancer url

https://ibmccmtsp.portsmouth.uk.ibm.com/CCMSTranslationService/jaxrs/translation/xml

Fill in the Content to Send field with the proper XML format for the operation you would like to execute.

Next click the POST button.

The request will be sent to the Translation Service server, and a response will be returned with the outcome of the request.
